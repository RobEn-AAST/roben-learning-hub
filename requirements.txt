# ==================================================
# DEVELOPMENT SETUP
# ==================================================

# 1. Install Node.js v20 or higher
# 2. Clone the repository
# 3. Run: npm install
# 4. Create .env.local with Supabase credentials
# 5. Run: npm run dev
# 6. Open: http://localhost:3000


# ==================================================
# LOCAL DEVELOPMENT WITH SUPABASE
# ==================================================

# Install Docker Desktop for Windows
# Install Supabase CLI: (use npx supabase)
# Initialize: npx supabase init
# Start local: npx supabase start
# Stop local: npx supabase stop
# Reset DB: npx supabase db reset
